Présentation du projet de PPD

Ce projet a été développé par des élèves du Master 1 Miage dans le cadre d'un projet tuteuré.
l'enjeu majeur de ce projet fut la manipulation d'une énorme quantité de données avec des outils de Big Data. Cette application web a pour objectif de présenter des données présentées sous format de statistiques et de graphiques. Ces données concernent les différentes courses des Taxis New Yorkais.

Les technologies utilisées

Pour développer cette application, nous avons utilisé les technologies suivantes :


Spark

Apache Spark est un moteur de traitement de données rapide dédié au Big Data. Il permet d’effectuer un traitement de larges volumes de données de manière distribuée (cluster computing).

Cassandra

Apache Cassandra est la base de données NoSQL la plus utilisée pour le stockage et le traitement de larges volumes de données. Dans le cadre de notre projet, nous l'avons utilisé pour le stockage de nos données brut

Python

Python est un langage de programmation interprété, multi-paradigme et multiplateforme. Il est le langage de programmation le plus utilisé dans le domaine du Machine Learning, du Big Data et de la Data Science.

Jupyter Notebook

Jupyter est une application web utilisée pour programmer dans plus de 40 langages de programmation, dont Python, Julia, Ruby, R, ou encore Scala

Hamachi

LogMeIn Hamachi est une application qui permet de créer un réseau privé virtuel assez simplement. Il nous a permis de connecter les machines de chaque membre du groupe sur un serveur commun

Angular

Angular est une plateforme de développement qui permet de créer des applications web dynamiques et immersives.
